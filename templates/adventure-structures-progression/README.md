# Adventure Mod With Structures And Progression

Use this template for jigsaw dungeons, discovered structures, dimension gates, unlock trees, and generated guide pages.

- Start with `nexusCreateJigsawStructure`.
- Add unlocks with `nexusCreateProgressionTree`.
- Use structure rule graphs for biome, height, distance, and nearby-structure predicates.
- Run `simulateNexusWorldgen`, `validateNexusProgression`, and `generateNexusGuideDrafts`.
