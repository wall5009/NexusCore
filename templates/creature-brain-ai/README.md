# Creature Mod With Brain AI

Use this template for mobs that need memories, sensors, schedules, behavior presets, and group behavior.

- Start with `nexusCreateBrainBasedMob`.
- Add data-driven spawn/render metadata with `nexusCreateDataDrivenEntity`.
- Use common memories, typed sensors, and `BehaviorLibrary` presets before writing custom behavior code.
- Run `validateNexusBrainAi` and entity GameTests.
