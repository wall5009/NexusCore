# Tech Mod With Multiblock Machines

Use this template for mods centered on machines, scalable tanks, ports, energy, fluids, and recipe processing.

- Start with `nexusCreateMultiblockMachine`.
- Add scalable storage with `nexusCreateScalableMultiblockTank`.
- Add recipe-family and recipe-transfer descriptors.
- Run `validateNexusMultiblocks`, `simulateNexusRecipes`, and `generateNexusBalanceDashboard`.
