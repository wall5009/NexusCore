# Automation And Logistics Mod

Use this template for pipes, cables, route planning, filters, persistence, chunk safety, and throughput simulation.

- Start with `nexusCreateAutomationNetwork`.
- Use `RoutePlanner` for priority, nearest-first, round-robin, fullness, demand, tag, and component-sensitive routing.
- Use `NetworkDiagnostics` for graph views, blocked routes, bottlenecks, and unloaded-route warnings.
- Run `simulateNexusRecipes` and `generateNexusBalanceDashboard`.
