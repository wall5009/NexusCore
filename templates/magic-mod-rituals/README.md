# Magic Or Ritual Mod

Use this template for altar crafting, summoning, weather rituals, portals, and claim-safe world interactions.

- Start with `nexusCreateRitualAltar`.
- Use `nexusCreateRitualWithMultiblock` when the ritual requires a structure.
- Keep dangerous effects behind `RitualSafetyPolicy.dangerousEffectsEnabled()`.
- Run `validateNexusRituals`, `generateNexusGuideDrafts`, and `simulateNexusRecipes`.
