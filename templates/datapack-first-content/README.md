# Datapack-First Content Mod

Use this template when most content starts as JSON definitions and Java is reserved for extension points.

- Start with `nexusCreateDataDefinition`.
- Add entities with `nexusCreateDataDrivenEntity`.
- Use `NexusDataDefinitions.content` for multiblocks, rituals, structures, progression, automation, and guide pages.
- Run `generateNexusSchemas`, `validateNexusMultiblocks`, `validateNexusRituals`, and `validateNexusProgression`.
