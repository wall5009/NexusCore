# Nexus Doctor and Debugging

`/nexus doctor` checks modules, content manifests, datagen, config graphs, network channels, and migration diagnostics.

Other commands:

- `/nexus version`
- `/nexus dump registries`
- `/nexus dump content`
- `/nexus dump config-schema`
- `/nexus dump packets`
- `/nexus validate assets`
- `/nexus validate tags`
- `/nexus validate recipes`
- `/nexus inspect held-item`
