# Getting Started with NexusCore v1.1

1. Apply the optional NexusCore Gradle plugin or copy a starter template.
2. Run `nexusSetupProject`.
3. Generate content with the scaffolding tasks.
4. Register content through `NexusRegistries`, `NexusItems`, `NexusBlocks`, or `NexusBlockSets`.
5. Generate data with `NexusData.plan(modid)`.
6. Run `runNexusValidation`.
7. Use `/nexus doctor` in a dev world before release.
