# Version Code

Use version folders for Minecraft-version-specific implementation:

```text
mc_1_20_1/src/main/java
mc_1_21_1/src/main/java
mc_26_1_2/src/main/java
```

Version-sensitive systems include registries, components, packet payloads, creative tabs, commands, datagen, item settings, worldgen, and menu/screen APIs.
